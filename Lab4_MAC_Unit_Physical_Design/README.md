# Lab 4: MAC Unit Physical Design

This folder contains a curated subset of the Lab 4 physical-design deliverables for the MAC unit.

## Included

- `docs/Lab4_Report.pdf`
- `sources/constraints/top.sdc`
- `sources/testbenches/tb_mac_unit.v`
- `artifacts/`
  - `MAC_TOP.v`
  - `MAC_TOP.sdf`
  - `MAC_TOP.drc.rpt`
  - `MAC_TOP.conn.rpt`
  - `MAC_TOP.antenna.rpt`
  - `MAC_TOP_preCTS.summary.txt`
  - `MAC_TOP_postCTS.summary.txt`
  - `MAC_TOP_postRoute.summary.txt`

## Omitted

- Foundry collateral and library data such as LEF, GDS, and cell-model files
- Innovus workspace state, command logs, and bulk timing archives
- Tutorial media and setup bundles
- Redundant or empty artifacts such as `mac_top_netlist.v` and the empty `power.rpt`

## Notes

- This folder is intended as a reference artifact set, not a full reproducible Innovus workspace.
- Final GDS and other technology-dependent deliverables are intentionally excluded from this public repository.
